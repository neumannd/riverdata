get.varmapping.hbm.netcdfR Documentation

Get mapping of "netCDF variables names for HBM river input files" to short names

Description

This mapping maps netCDF variable names as used in HBM river input files to corresponding short names, which are often used.

Usage

get.varmapping.hbm.netcdf()

Value

nested lists; look at the output yourself

Author(s)

Daniel Neumann, daniel.neumann@io-warnemuende.de

See Also

write.river.netCDF

Examples


  # get variable mapping
  varmapping <- get.varmapping.hbm.netcdf()
